# Burn and Turn Poker Trainer - Website

Official website for the Burn and Turn Poker Trainer mobile app.

## About

This website provides:
- Privacy Policy (required for App Store & Google Play)
- Terms of Service (required for app stores)
- Support/Contact page
- App information and features

## Tech Stack

- Next.js 15.1.4
- React 19.0.0
- Static export for Vercel deployment

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Creates a static export in the `out` directory.

## Deployment

Deployed on Vercel. Simply push to main branch for automatic deployment.

## URLs

Once deployed, the following URLs will be available:
- Homepage: https://[your-domain].vercel.app
- Privacy Policy: https://[your-domain].vercel.app/privacy
- Terms of Service: https://[your-domain].vercel.app/terms
- Support: https://[your-domain].vercel.app/support

These URLs should be used in:
- App Store Connect (Privacy Policy URL, Support URL)
- Google Play Console (Privacy Policy URL, Website URL)
